How does javascript work?

Javascript is an interpreted language.The javascript engine converts the code written in javascript line by line.It is also called a client side scripting language.In web browser like chrome,edge,safari to support javascript code they use their javascript engine.For example chrome browser has its own javascript engine which is called V8.When we run a javascript program inside a browser,the browser's engine receive the code and runs the source code through several steps to obtain the output.

Difference between local storage and session storage?

local storage:

  • local storage stores data on a permanent basis
  • local storage stores data for a single browser or tab.
  • The amount of data in local storage is 5-10 Mbs(browser dependent)
  • In local storage when a user will close the browser stored data will be available for day,week and month.

session storage:

  • session storage stores data on a temporary basis.
  • session storage data can be accessiable from all windows.
  • The amount of data in local storage is 5 Mbs
  • In session storage when a user will close the browser stored data will be deleted.
